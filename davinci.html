Media Page

Overview

The Media Page is your central hub for importing, organizing, and managing all media assets used in your DaVinci Resolve project. This is where your video editing journey begins.

üìñ Tutorial: Importing Media

  1. Open DaVinci Resolve and create a new project or open an existing one
  2. Navigate to the Media Page by clicking the "Media" tab at the bottom
  3. In the Media Storage panel (top-left), browse to the folder containing your media files
  4. Select the files you want to import and drag them into the Media Pool
  5. Alternatively, right-click in the Media Pool and select "Import Media" to browse files
  6. Your imported media now appears in the Media Pool, ready for editing

Key Features & Tools

üóÇÔ∏è Creating and Using Bins

  1. Right-click in the Media Pool and select "New Bin"
  2. Name your bin (e.g., "Interviews", "B-Roll", "Music")
  3. Drag and drop media files into the appropriate bins
  4. Create nested bins by dragging one bin into another for complex organization
  5. Use color-coded bins by right-clicking and selecting a color

Media Pool Functions

  • Import video, audio, and image files to the Media Pool
  • Create bins to organize media into folders for easy access
  • View and edit metadata (file name, duration, creation date)
  • Smart Bins automatically organize media based on metadata criteria
  • Preview media by playing back clips directly within the Media Pool

Clip Management

  • Modify clip properties (frame rate, pixel aspect ratio, color space)
  • Add clips to timeline using Insert, Overwrite, and Append modes
  • Create markers on clips for reference or notes
  • Review content before adding to timeline

üéØ Using Smart Bins

  1. Right-click in the Media Pool and select "Add Smart Bin"
  2. Set criteria like "Frame Rate equals 60 fps" or "Duration greater than 10 seconds"
  3. Click OK - the Smart Bin automatically populates with matching clips
  4. Smart Bins update automatically as you add new media to your project

Cut Page

Overview

The Cut Page is a streamlined editing environment focused on quick cuts and assembly. It's designed for rapid editing workflows and getting your rough cut together efficiently.

‚ö° Quick Start: Fast Editing Workflow

  1. Switch to the Cut Page by clicking the "Cut" tab at the bottom
  2. Your media appears in the Source Tape viewer at the top
  3. Scrub through the Source Tape to find the clip you want
  4. Mark In (I key) and Out (O key) points around the section you want to use
  5. Click "Append" to add the clip to the end of your timeline
  6. Repeat this process to quickly build your rough cut

Cut Tools & Techniques

‚úÇÔ∏è Using the Blade Tool

  1. Press B on your keyboard to activate the Blade Tool
  2. Position the playhead where you want to make a cut
  3. Click on the clip to split it at the playhead position
  4. The Blade Tool cuts across all tracks at once, maintaining sync
  5. Press A to return to the Selection Tool

Source Tape

  • View all clips as continuous footage
  • Quickly scrub through all media
  • Mark In/Out points for editing
  • Fast review without switching clips

Dual Timeline

  • Overview timeline shows entire edit
  • Detailed timeline for precise editing
  • Zoom in/out seamlessly
  • Navigate quickly between sections

üé¨ Three-Point Editing

  1. Mark In and Out points on your source clip (2 points)
  2. Position the playhead in your timeline where you want the clip (1 point)
  3. Press F9 for Insert edit (shifts existing clips) or F10 for Overwrite (replaces clips)
  4. The clip duration is determined by the source In/Out points

Edit Page

Overview

The Edit Page is the primary non-linear editing (NLE) environment for detailed timeline manipulation. This is where you refine your edit with precision and creativity.

üéØ Mastering Trim Modes

  1. Ripple Trim (Y): Hover over the edge of a clip until you see the yellow bracket icon
  2. Click and drag to adjust the clip length - subsequent clips shift automatically
  3. Roll Trim (U): Position cursor between two clips until you see the red bracket icon
  4. Drag to adjust both clips simultaneously without changing overall timeline length
  5. Slip Tool (Y twice): Changes what part of a clip plays without affecting timing
  6. Slide Tool (Y three times): Moves clip along timeline, adjusting adjacent clips

Advanced Editing Tools

üîß Using the Transform Tool

  1. Select a clip in the timeline
  2. Open the Inspector panel (top-right corner)
  3. Navigate to the "Transform" section
  4. Adjust Position X/Y to move the clip on screen
  5. Modify Scale to zoom in or out
  6. Change Rotation to tilt the clip
  7. Adjust Anchor Point to change the center of rotation/scaling
  8. Enable keyframes to animate these properties over time

Trim Tools

  • Ripple Trim: Adjusts duration and shifts subsequent clips
  • Roll Trim: Adjusts adjacent clip edges simultaneously
  • Slip Tool: Change in/out points, keep duration
  • Slide Tool: Move clip, affect adjacent clips

Timeline Controls

  • Track enable/disable, lock, and solo
  • Zoom controls for detail and overview
  • Snapping for precise alignment
  • Linked selection for synchronized editing

Effects Library

  • Video transitions (dissolves, wipes)
  • Audio transitions (crossfades)
  • Color effects and filters
  • Blur, glow, and stylize effects

Inspector Panel

  • Transform controls (position, scale, rotation)
  • Cropping and zoom options
  • Blend modes and opacity
  • Speed/duration adjustments

üé® Applying Video Transitions

  1. Open the Effects Library panel (top-left)
  2. Navigate to "Video Transitions" and expand the category
  3. Drag a transition (e.g., "Cross Dissolve") onto the cut between two clips
  4. Double-click the transition in the timeline to open in the Inspector
  5. Adjust duration by dragging the edges of the transition
  6. Customize parameters in the Inspector (ease, alignment)

‚è±Ô∏è Speed Changes & Retiming

  1. Right-click on a clip and select "Retime Controls"
  2. For constant speed: Click the speed indicator and choose a percentage (50% = half speed)
  3. For smooth slow motion: Right-click > Retime Curve > Speed Ramp
  4. Add keyframes to create speed variations within a single clip
  5. Use "Optical Flow" in the Inspector for smoother slow motion

Fusion Page

Overview

The Fusion Page is a node-based compositing environment for creating visual effects and motion graphics. It provides Hollywood-grade VFX capabilities within DaVinci Resolve.

üåü Creating Your First Fusion Composition

  1. Select a clip in the Edit page timeline
  2. Switch to the Fusion page - a MediaIn and MediaOut node appear automatically
  3. Right-click in the node area and select "Add Tool" or press Shift+Space
  4. Search for a tool (e.g., "Text+") and press Enter to add it
  5. Connect the Text+ node between MediaIn and MediaOut by dragging
  6. Modify the text in the Inspector panel on the right
  7. Return to Edit page - your text now appears on the clip

Fusion Tools & Techniques

üé≠ Green Screen Keying

  1. Add your green screen footage to the timeline and open in Fusion
  2. Add a "Delta Keyer" node (Shift+Space, search "Delta Keyer")
  3. Connect MediaIn to Delta Keyer to MediaOut
  4. Use the eyedropper tool to sample the green color
  5. Adjust "Low" and "High" sliders to refine the key
  6. Add a "Merge" node to composite over a background
  7. Connect your background to the Merge node's background input
  8. Fine-tune with "Erode/Dilate" and "Blur" for clean edges

Node-Based Workflow

  • Visual node tree for effect composition
  • Non-destructive layer-based effects
  • Connect nodes to create complex chains
  • Reusable effect templates

VFX Tools

  • 3D compositing and tracking
  • Keying and rotoscoping
  • Particle systems
  • Motion graphics animation

Text & Titles

  • Text+ tool for animated titles
  • 3D text with depth and lighting
  • Follow path animation
  • Custom fonts and styling

Motion Graphics

  • Shape tools for graphics creation
  • Spline-based animation
  • Modifiers for complex motion
  • Expressions for mathematical animation

‚ú® Animated Lower Third Title

  1. Create a new Fusion composition or select a clip
  2. Add a "Background" node and set the color to transparent or your choice
  3. Add a "Rectangle" node and position it in the lower third of frame
  4. Add a "Text+" node for the name/title
  5. Use "Merge" nodes to combine the rectangle and text
  6. Animate the position: Go to frame 0, set X position off-screen, add keyframe
  7. Move to frame 30, set X position to final location, add keyframe
  8. Apply easing in the Spline editor for smooth motion

Color Page

Overview

The Color Page is dedicated to color correction and grading, providing professional-grade tools used in Hollywood productions for achieving the perfect look.

üé® Basic Color Correction Workflow

  1. Switch to the Color page - your timeline appears at the bottom
  2. Select the first clip you want to grade
  3. Start with "Primary Color Wheels" in the bottom panel
  4. Balance the image: Adjust "Lift" for shadows, "Gamma" for midtones, "Gain" for highlights
  5. Use the center wheel to adjust color temperature
  6. Check the scopes (right panel) to ensure proper exposure
  7. Apply a LUT if desired (Right-click on the node > LUT > Browse)
  8. Add creative grading with additional nodes (Right-click in node tree > Add Node)

Color Grading Tools

üîß Using Color Wheels Effectively

  1. Lift (Shadows): Click and drag the center circle towards cool colors (blue) to remove warm color cast from shadows
  2. Adjust the slider below to control shadow brightness
  3. Gamma (Midtones): Drag towards warm colors (orange) for golden hour look
  4. Use slider to brighten or darken overall image
  5. Gain (Highlights): Subtle adjustments for sky and bright areas
  6. Offset: Shifts all tonal ranges uniformly for overall color tint
  7. Reset individual wheels by clicking the reset button in the center

Color Wheels

  • Lift: Shadows/darkest parts
  • Gamma: Midtones adjustment
  • Gain: Highlights/brightest parts
  • Offset: Overall color balance

Curves

  • Luma vs. Sat (saturation control)
  • Luma vs. Luma (contrast)
  • Hue vs. Sat (desaturate colors)
  • Hue vs. Hue (color shifting)

Scopes

  • Waveform: Luminance levels
  • Parade: RGB channels separately
  • Vectorscope: Hue and saturation
  • Histogram: Luminance distribution

Qualifiers

  • HSL qualifier for color selection
  • Luma qualifier for brightness ranges
  • Power Windows for spatial masks
  • Tracking for moving objects

üéØ Skin Tone Correction

  1. Add a new node (Right-click in node tree > Add Node > Add Serial)
  2. Open the "Qualifier" panel
  3. Use the eyedropper to sample skin tones in the viewer
  4. Adjust HSL ranges to refine the selection (check with the highlight mode)
  5. In the Color Wheels, gently shift Gamma towards natural skin tone (slightly orange)
  6. Reduce saturation slightly if needed for natural look
  7. Soften the mask edges using "Blur" in the Qualifier panel

üåÖ Creating Cinematic Look

  1. Balance your footage first (neutral correction)
  2. Add a new serial node for creative grading
  3. In Color Wheels: Push Lift slightly towards teal/cyan
  4. Push Gain slightly towards orange/yellow
  5. Reduce saturation in midtones slightly (Gamma wheel slider)
  6. Add another node for contrast: Create an S-curve in the Curves panel
  7. Fine-tune with a LUT for the final look
  8. Apply to similar shots using "Still" (gallery) or "Apply Grade"

üìä Reading Scopes

  1. Waveform: Bottom = black (0 IRE), Top = white (100 IRE). Keep within legal range
  2. Parade: RGB should be balanced. If red is higher, image is too warm
  3. Vectorscope: Skin tones fall along the line between yellow and red
  4. Histogram: Should have data across the range without clipping at either end

Fairlight Page

Overview

The Fairlight Page is a full-featured digital audio workstation (DAW) for audio mixing, editing, and mastering. It provides professional audio post-production capabilities.

üéµ Basic Audio Mixing Workflow

  1. Switch to the Fairlight page - your timeline audio tracks appear
  2. Open the Mixer panel (if not visible, click the Mixer button)
  3. Set levels: Dialogue should peak around -12dB to -6dB
  4. Music should be lower, around -18dB to -12dB
  5. Sound effects vary but generally -20dB to -12dB
  6. Use the faders to adjust overall track volume
  7. Pan tracks left or right for stereo width
  8. Add EQ and compression as needed for clarity

Audio Tools & Effects

üéöÔ∏è Applying EQ (Equalization)

  1. Select an audio track in the Fairlight timeline
  2. In the Mixer, click the "EQ" button on the channel strip
  3. The EQ panel opens showing a frequency spectrum
  4. For dialogue: Reduce low frequencies below 80Hz (remove rumble)
  5. Boost slightly around 3-5kHz for clarity and presence
  6. Cut harsh frequencies around 6-8kHz if needed (sibilance)
  7. Use high-pass and low-pass filters to remove unwanted frequencies
  8. A/B compare by toggling the EQ on/off

Mixer Controls

  • Faders: Adjust track volume
  • Pan: Stereo positioning
  • Solo/Mute: Isolate or silence tracks
  • Meters: Visual level monitoring

Audio Effects

  • EQ for frequency shaping
  • Compression for dynamics control
  • De-esser for harsh 'S' sounds
  • Reverb and delay for space

Editing Tools

  • Elastic wave retiming
  • Audio normalization
  • Clip gain adjustments
  • Crossfades and automation

Advanced Features

  • ADR and dialogue replacement
  • Multi-track recording
  • Fairlight FX plugins
  • Bussing and routing

üé§ Noise Reduction

  1. Select the audio clip with unwanted background noise
  2. Go to the Effects Library and find "Voice Isolation" or "Noise Reduction"
  3. Drag the effect onto your audio clip
  4. In the Inspector, adjust the threshold to remove noise without affecting dialogue
  5. For Dialogue Cleaner: Set the amount based on noise level (start around 50%)
  6. Fine-tune with the "Reduce" slider
  7. Use "Learn" feature if available to sample the noise floor

üîä Audio Compression Basics

  1. In the Mixer, click "Dynamics" on your dialogue track
  2. Select "Compressor" from the dropdown
  3. Set Threshold to where your audio consistently hits (around -18dB)
  4. Set Ratio to 3:1 for gentle compression (good starting point)
  5. Adjust Attack to 10-30ms (how quickly compression engages)
  6. Set Release to 50-100ms (how quickly it lets go)
  7. Use the Gain/Makeup to compensate for reduced volume
  8. Watch the gain reduction meter - aim for 3-6dB of reduction

üéº Mixing Music with Dialogue

  1. Place music on a separate track from dialogue
  2. Lower music volume when dialogue is present (ducking)
  3. Method 1: Manually keyframe volume down during dialogue sections
  4. Method 2: Use sidechain compression (dialogue triggers music reduction)
  5. Set music to peak around -18dB to -12dB when alone
  6. Duck to -25dB to -20dB during critical dialogue
  7. Use crossfades on music edits for smooth transitions

Deliver Page

Overview

The Deliver Page is used for exporting your final project in various formats. It provides comprehensive rendering options for different platforms and requirements.

üì§ Export for YouTube (1080p)

  1. Switch to the Deliver page
  2. In the Render Settings panel (left side), select "YouTube" from the preset dropdown
  3. Set Format to "MP4" and Codec to "H.264"
  4. Set Resolution to "1920x1080 HD"
  5. Quality: Choose "Restrict to" and set bitrate to 10,000 Kb/s for high quality
  6. Under Audio, ensure codec is "AAC" at 320 Kb/s
  7. Name your file and choose export location at the bottom
  8. Click "Add to Render Queue" then click "Start Render"

Export Settings & Formats

üé¨ Export for Professional Use (ProRes)

  1. Select "Custom" from the format dropdown
  2. Set Format to "QuickTime" and Codec to "Apple ProRes 422"
  3. Choose resolution based on source (4K, 1080p, etc.)
  4. Set frame rate to match your timeline (23.98, 24, 29.97, etc.)
  5. Audio: Set to "Linear PCM" 48kHz 24-bit for highest quality
  6. This creates large files but maintains maximum quality
  7. Ideal for further editing or professional broadcast

Common Export Formats

  • H.264 MP4: Web, social media, general viewing
  • H.265/HEVC: Smaller files, newer devices
  • ProRes: Professional editing and archival
  • DNxHD/DNxHR: Avid workflows

Platform Presets

  • YouTube (various qualities)
  • Vimeo optimized settings
  • Instagram posts and stories
  • TikTok and vertical video

Quality Settings

  • Bitrate: Higher = better quality, larger file
  • Constant/Variable: CBR vs VBR encoding
  • Quality vs Size: Balance for intended use
  • Two-pass: Better quality, longer render

Advanced Options

  • Custom LUTs for delivery
  • Burn-in timecode and metadata
  • Multiple format batch exports
  • Render queue management

üì± Export for Instagram (Square and Stories)

  1. For Square Posts: Set resolution to 1080x1080
  2. For Stories: Set resolution to 1080x1920 (9:16 aspect ratio)
  3. Use H.264 codec with high-quality settings
  4. Set bitrate to 6,000-8,000 Kb/s for good quality
  5. Keep file size under 4GB for Instagram compatibility
  6. Audio: AAC codec, 128-320 Kb/s
  7. For Stories: Keep text and important content in the safe zone

‚ö° Render Queue Management

  1. Set up multiple exports with different settings
  2. Click "Add to Render Queue" for each export instead of rendering immediately
  3. View all queued jobs in the "Render Queue" tab
  4. Reorder jobs by dragging them up or down
  5. Pause, resume, or cancel individual jobs
  6. Set up overnight rendering for large projects
  7. Jobs will render sequentially when you click "Start Render"

üéØ Optimizing Render Settings

  1. For Web: Use H.264, variable bitrate, two-pass encoding
  2. File Size Priority: Lower bitrate, use H.265 if supported
  3. Quality Priority: Higher bitrate, ProRes for mastering
  4. Speed Priority: Use GPU acceleration, single-pass encoding
  5. Test short segments before rendering full projects
  6. Consider your audience's viewing devices and bandwidth
